Tags do Git

```powershell
# Lista tags
git tag

# Lista com "regex"
git tag -l "v1.8.5*"

# Adiciona tag
git tag -a v1.4 -m "my version 1.4"

#
git show v1.4

# Adiciona tag
git tag v1.4-lw

# Logs
git log --pretty=oneline

# Push tag para o repo
git push origin v1.5

# Marcar "compromissos espec√≠ficos"
git tag 1.1.0 <insert-commitID-here>

# Push all tags para o repo
git push --tags
```
